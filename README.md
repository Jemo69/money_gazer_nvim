# Moneygazer Nvim Theme

A dark, high-contrast Neovim theme with gold, purple, and violet accents.

## Installation

### lazy.nvim

```lua
{
  dir = "/path/to/Moneygazer/nvim", -- Adjust path if needed
  name = "moneygazer",
  config = function()
    vim.cmd.colorscheme("moneygazer")
  end
}
```

## Features
- Full support for TreeSitter
- Native LSP support
- Dark blue background with high readability
